一、核心上下文层（自动加载）

| 层级     | 文件             | 位置                | 作用                                                 |
| -------- | ---------------- | ------------------- | ---------------------------------------------------- |
| 全局     | CLAUDE.md        | ~/.claude/CLAUDE.md | 适用于所有项目的用户级默认指令                       |
| 项目     | CLAUDE.md        | ./CLAUDE.md         | 项目级上下文：构建命令、架构、注意事项，纳入版本控制 |
| 本地覆盖 | .claude.local.md | ./.claude.local.md  | 个人偏好，不共享给团队，应加入 .gitignore            |
| 子目录   | CLAUDE.md        | ./**/CLAUDE.md      | monorepo 模块级上下文，按需加载                      |

加载优先级：.claude.local.md > ./CLAUDE.md > ~/.claude/CLAUDE.md（逐节覆盖，而非整体替换）

二、条件与规则层（按需加载）

| 层级    | 文件                       | 作用                                                         |
| ------- | -------------------------- | ------------------------------------------------------------ |
| 规则    | .claude/rules/*.md         | 带 paths: frontmatter，仅在编辑匹配文件时加载（如 src/**/*.ts） |
| Hookify | .claude/hookify.*.local.md | 监控特定工具调用模式（bash/file/stop/prompt），匹配时触发 warn 或 block |

三、功能扩展层（主动调用或分派）

| 层级     | 文件                                 | 作用                                                         |
| -------- | ------------------------------------ | ------------------------------------------------------------ |
| 技能     | .claude/skills/<name>/SKILL.md       | 打包的专业知识/工作流，可被 Claude 自动激活或用户通过 /name调用 |
| 子代理   | .claude/agents/*.md                  | 定义专用自主子代理，用于复杂多步任务，有独立子进程           |
| 斜杠命令 | .claude/commands/*.md                | 用户自定义 /command-name，通过 / 触发                        |
| 插件     | 插件目录下 commands/ agents/ skills/ | 与上述相同的结构，作用于插件命名空间                         |

四、持久化记忆层

| 层级     | 文件                                          | 作用                                                    |
| -------- | --------------------------------------------- | ------------------------------------------------------- |
| 记忆索引 | ~/.claude/projects/<project>/memory/MEMORY.md | 跨会话持久化记忆的索引文件，上限 25KB/200 行            |
| 记忆条目 | memory/*.md                                   | 按类型（user/feedback/project/reference）存储的记忆内容 |

   前端项目的husky + **pre-commit**, 可以实现上传前的代码检查, dotnet-husky，java也有但似乎不必要

5. MCP

   MCP简单的理解是第三方提供的接口(不是专指REST API), 此处的第三方也非专指通过网络访问的网站, 它只是工具的一个集合, 不应该包含细节

   常见的MCP包括:(1) 文件系统, EXCEL, 数据库 (2) GIT

   常见的MCP方式包括: (1) 本地进程（stdio）（npx最典型） (2) 远程服务 (HTTP/Streamable HTTP)

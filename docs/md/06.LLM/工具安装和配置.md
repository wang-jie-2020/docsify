- CC配置deepseek

```bash
vim ~/.bashrc

export ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic
export ANTHROPIC_AUTH_TOKEN=${DEEPSEEK_API_KEY}
export API_TIMEOUT_MS=600000
export ANTHROPIC_MODEL=deepseek-chat
export ANTHROPIC_DEFAULT_HAIKU_MODEL=deepseek-chat
export CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1

source ~/.bashrc
```



## 其他工具

### LiteLLM

这个工具主要的功能:

(1) 统一网关, 兼容不同AI供应商的LLM-API(默认0.0.0.0:4000) 

(2) 隐藏真实的KEY

(3) 管理不同Team、不同个人的虚拟KEY, 可以限制TOKEN数量或者MODEL

(4) MCP、ACP等等未尝试

官方提供了镜像(见部署仓库), DEBUG下只在WSL通过Pip安装

```bash
pip install 'litellm[proxy]'
nohup litellm --config config.yaml
```

```yaml
model_list:
  # ===== DeepSeek =====
  - model_name: deepseek-chat
    litellm_params:
      model: deepseek/deepseek-chat
      api_key: os.environ/DEEPSEEK_API_KEY

  - model_name: deepseek-reasoner
    litellm_params:
      model: deepseek/deepseek-reasoner
      api_key: os.environ/DEEPSEEK_API_KEY

general_settings:
  master_key: sk-my-local-key      # 访问本地网关用的统一 key，自定义
  # 可选：开启管理后台
  # ui_username: admin
  # ui_password: admin123

```

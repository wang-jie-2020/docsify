## 🖥️ 系统操作

| 快捷键      | 功能                |
| ----------- | ------------------- |
| `⌘ + Space` | 打开 Spotlight 搜索 |
| `⌘ + Tab`   | 切换应用            |
| `⌘ + ~`     | 同一应用内切换窗口  |
| `⌘ + Q`     | 退出应用            |
| `⌘ + W`     | 关闭窗口/标签页     |
| `⌘ + M`     | 最小化窗口          |
| `⌘ + H`     | 隐藏当前应用        |
| `⌘ + ,`     | 打开当前应用设置    |
| `⌃ + ⌘ + Q` | 锁屏                |
| `⇧ + ⌘ + 3` | 全屏截图            |
| `⇧ + ⌘ + 4` | 区域截图            |
| `⇧ + ⌘ + 5` | 截图/录屏工具       |



## 📝 文本编辑

| 快捷键          | 功能               |
| --------------- | ------------------ |
| `⌘ + C / V / X` | 复制 / 粘贴 / 剪切 |
| `⌘ + Z`         | 撤销               |
| `⌘ + ⇧ + Z`     | 重做               |
| `⌘ + A`         | 全选               |
| `⌘ + F`         | 查找               |
| `⌘ + G`         | 查找下一个         |
| `⌥ + ←/→`       | 按词跳转           |
| `⌘ + ←/→`       | 跳到行首/行尾      |
| `⌘ + ↑/↓`       | 跳到文档开头/结尾  |
| `⌥ + Delete`    | 删除前一个词       |
| `⌘ + Delete`    | 删除到行首         |



## 🌐 浏览器（Safari / Chrome）

| 快捷键            | 功能              |
| ----------------- | ----------------- |
| `⌘ + T`           | 新建标签页        |
| `⌘ + R`           | 刷新页面          |
| `⌘ + L`           | 聚焦地址栏        |
| `⌘ + D`           | 收藏书签          |
| `⌘ + ⇧ + T`       | 恢复关闭的标签页  |
| `⌘ + [` / `⌘ + ]` | 前进/后退         |
| `⌘ + 数字`        | 切换到第 N 个标签 |



## 📁 Finder 文件管理

| 快捷键           | 功能              |
| ---------------- | ----------------- |
| `⌘ + N`          | 新建 Finder 窗口  |
| `⌘ + ⇧ + N`      | 新建文件夹        |
| `Space`          | 快速预览文件      |
| `⌘ + Delete`     | 移到废纸篓        |
| `⌘ + ⇧ + Delete` | 清空废纸篓        |
| `⌘ + I`          | 显示文件信息      |
| `⌘ + ⇧ + .`      | 显示/隐藏隐藏文件 |
| `⌘ + ↑`          | 前往上级目录      |

## VOLTA

```bash
curl https://get.volta.sh | bash

volta install node@22.5.1
volta install node

volta pin node@20.16
```

## WebStorm 中跳转问题

在项目根目录下[文件：jsconfig.json]

```js
{
  "compilerOptions": {
    "baseUrl": "./",
    "paths": {
      "@/*": [
        "src/*"
      ]
    }
  },
  "exclude": [
    "node_modules",
    "dist"
  ]
}
